# mern-project
Este repositorio es la estructura base del proyecto MERN disponible para que todos puedan colaborar

MongoDB
React
Node.js
Express.js

frontend - React
1. Copiar '.env.example' a '.env'
2. Configurar la URL del backend

backend - API REST
1. Copiar '.env.example' a '.env'
2. Configurar variables necesarias


EJECUCION LOCAL

Backend:
cd backend
npm install
npm run dev

Frontend:
cd frontend
npm install
npm run dev
